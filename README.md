# Bigecho

Bigecho は受け取った文字列を大きな文字で表示するソフトウェアです。

# Release

リリースは[こちら](https://github.com/aho4ahoaho/Bigecho/releases)からダウンロードできます。

# Usage

Bigecho を実行すると、標準入力から文字列を受け取り、標準出力に大きな文字で表示します。

```bash
$ bigecho Hello, World!
```
